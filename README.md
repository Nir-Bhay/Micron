# Micron – Animated UI Component Library for Next.js

## Project Title
Micron

## Tech Stack
- **Framework**: Next.js (React) 16
- **Language**: TypeScript
- **Styling**: Tailwind CSS, Shadcn UI utilities
- **Animations**: Framer Motion, custom React components
- **Build Tools**: Vite, ESLint, Prettier

## Installation & Usage
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Nir-Bhay/Micron.git
   cd Micron
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the demo site**:
   ```bash
   npm run dev
   ```
   Open `http://localhost:3000` to explore the components.
4. **Add to your project**:
   ```bash
   npm i micron
   ```
   Then import the desired components, e.g.:
   ```tsx
   import { AnimatedCircularProgressBar } from "micron/components/ui";
   ```
5. **Build for production**:
   ```bash
   npm run build
   ```

## Contribution Guidelines
- Fork the repo and create a feature branch (`git checkout -b feature/your-feature`).
- Follow the existing TypeScript and ESLint configurations.
- Write tests for new components and ensure coverage.
- Document any new component in the `README.md` with usage examples.
- Submit a pull request with a clear change description.
- Keep commits atomic and descriptive.
